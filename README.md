# Hearthstone Elopements Website

This repository contains the website for **Hearthstone Elopements**, a small elopement ceremony venue located in West Virginia.

The site is a simple static website built with HTML, CSS, and JavaScript and hosted using **GitHub Pages**.

## Live Website
https://lwright0695.github.io/hearthstone-elopements/

## Project Goals
- Provide a simple landing page for couples seeking elopement ceremonies
- Capture leads through an availability form
- Demonstrate marketing analytics and automation concepts

## Tech Stack
- HTML
- CSS
- JavaScript
- GitHub Pages

## Future Improvements
